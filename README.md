# FitVatch - Mockup Funcional

Aplicación web de fitness y atletismo construida con React + TypeScript.

## 🚀 Empezar Rápido

### Instalación
```bash
npm install
```

### Desarrollo
```bash
npm run dev
```
Abre http://localhost:5173 en tu navegador.

### Build para Producción
```bash
npm run build
```
